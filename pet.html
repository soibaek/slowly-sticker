🐾 桌宠助手 · 使用说明

这是一个像素风桌面小动物原型。小动物没有任何外框，可以自由移动。

  1. 拖拽：按住小动物拖动，松手即放置到新位置。
  2. 点击：单击小动物，旁边弹出互动菜单（4 个选项）。
  3. 陪我聊聊：展开「随便聊聊 / 安慰我一下」，底部出现输入框（接入 AI API 处已注明）。
  4. 一起玩：根据当前动物弹出对应玩具动画（猫=逗猫棒、狗=飞盘、兔=草球…）。
  5. 摸摸：光标变手型，小动物转圈圈 360°。
  6. 去睡觉：小动物蜷缩入睡、停止移动；再次点击唤醒。
  7. 🎲 新安置：随机移动到屏幕某处（左上 / 中央 / 右下…）。
  8. 🐾 自动巡游：开启后小动物自动缓慢漫步。
  9. 底部缩略图：切换不同动物（玩具会随之变化）。

气泡里的「此处接入 AI API」表示：实际对话回复需要在此调用大模型接口。
