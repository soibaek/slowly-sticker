提醒功能测试工具

用于测试本地提醒功能和 Notification API

环境检测

手机端兼容性检查

手机浏览器兼容性说明：

  • iOS Safari (14.5+): 支持 Notification，但需要用户在页面中主动触发权限请求
  • Android Chrome: 支持 Notification，但部分版本需要 HTTPS
  • 微信内置浏览器: 不支持 Notification，会使用 alert() fallback
  • 其他不支持的浏览器: 自动降级为 confirm() 或 alert()

快速测试

提示：为了快速测试，可以将分钟数改为 5（5秒后触发）

测试用例

点击测试用例可直接执行：

[成功] 正常提醒 - 提醒我10分钟后喝水
[成功] 简化格式 - 5分钟后提醒我开会
[成功] 仅时间格式 - 30分钟后休息
[警告] 缺少时间 - 提醒我喝水 (应提示格式错误)
[正常] 非提醒内容 - 你好呀，今天天气怎么样？ (应调用AI)
[快速] 5秒测试 - 提醒我5秒后喝水

实时日志

日志颜色说明

V
SUCCESS (绿色): 操作成功、提醒设置完成、通知已发送
i
INFO (蓝色): 一般信息、调试信息
!
WARNING (橙色): 警告信息、需要用户授权、使用 fallback
X
ERROR (红色): 错误信息
T
TRIGGER (紫色): 提醒触发
